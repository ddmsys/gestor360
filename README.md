# Site OGestor360®

Site Gestor 360

## Posicionamento do site

O site não é a página de um livro. É o hub de um método — o Gestor360® como sistema vivo de liderança. O livro é o ponto de entrada. As ferramentas são a razão para o leitor voltar. A mentoria e os treinamentos são a conversão de longo prazo. A comunidade é o que transforma compradores em defensores da marca.

## Arquitetura de páginas

### 01 Home

Frase de impacto do método + chamada para o livro + acesso rápido às ferramentas + depoimentos do lançamento + vídeo dos autores. Objetivo: converter visitante em lead ou comprador em menos de 60 segundos.

### 02 O Livro

Página dedicada ao Manual do Gestor360®: sinopse, estrutura dos 10 capítulos, amostra do conteúdo, fotos do lançamento, onde comprar (livraria física, site, Amazon, Mercado Livre). Badge "Lançado em [data] — +400 presentes".

### 03 Ferramentas

As 31 ferramentas em formato A4, organizadas por capítulo. Acesso via cadastro gratuito (e-mail + WhatsApp). Leitores com código do livro desbloqueiam pacote premium. Esta página alimenta o mailing e o WhatsApp da DDM.

### 04 O Método

O Gestor360® explicado como sistema: as 3 dimensões (técnica + neurociência + consciência), os 10 capítulos como jornada, o ciclo dos 90 dias. Prepara o terreno para mentoria e treinamentos corporativos.

### 05 Mentoria & Treinamentos

Formatos disponíveis: palestras, workshops, mentoria individual, licenciamento do método para consultores. Formulário de interesse. Pode começar simples (uma seção) e crescer para página própria.

### 06 Sobre

Flávio Di Morais e Marcelo Caetano — histórias reais, trajetórias, o que os une. Foto do lançamento. Links para redes sociais. @oCaraDoLivro em destaque para Flávio.

### 07 Blog

Criado agora, alimentado depois. Estrutura pronta com categorias: Liderança, Finanças, Delegação, Mentalidade, Cases. Cada post pode ser adaptado das seções do livro — conteúdo já existe, é só distribuir.

### 08 Comunidade (fase 2)

Área exclusiva para leitores com código do livro: fórum, sessões ao vivo mensais, perguntas aos autores. Pode começar como grupo de WhatsApp gerenciado pelo site antes de virar área fechada.

## Fluxo de captação de leads

### 1 Leitor escaneia QR Code do livro

Cada capítulo tem um QR Code que aponta para a página das ferramentas daquele capítulo no site.

### 2 Cadastro rápido: nome + e-mail + WhatsApp

Formulário de uma página. Campos obrigatórios: nome e e-mail. WhatsApp opcional mas incentivado ("receba notificações de novas ferramentas pelo WhatsApp"). Ao cadastrar, aceita receber conteúdo do Gestor360®.

### 3 Acesso imediato às ferramentas gratuitas

Ferramentas básicas (3 por capítulo) liberadas no cadastro. As 31 ferramentas completas só para quem insere o código do livro.

### 4 E-mail de boas-vindas automático

Sequência de 3 e-mails: boas-vindas + link das ferramentas → dica da semana baseada no livro → convite para mentoria ou treinamento.

### 5 WhatsApp para quem optou

Mensagem de boas-vindas + link para grupo exclusivo de leitores. Base separada por capítulo de origem (sinal de interesse específico — útil para segmentar campanhas futuras).

## Tecnologia recomendada

### Plataforma

WordPress + Elementor Pro — ou Webflow se tiver dev

### E-mail marketing

Mailchimp (gratuito até 500 contatos) → RD Station quando escalar

### WhatsApp

Zapier conectando formulário → WhatsApp Business API (Twilio ou Z-API)

### Ferramentas (PDFs)

Hospedadas no Google Drive ou AWS S3 — links protegidos por token

### Formulários e leads

Typeform ou formulário nativo do WordPress com integração ao Mailchimp

### Analytics

Google Analytics 4 + Meta Pixel — medir origem de cada lead por QR Code

### QR Codes

Um QR Code por capítulo com UTM rastreável — saber qual capítulo gera mais acesso

### Domínio sugerido

ogestor360.com

## Roadmap de lançamento

### Fase 1 — Semanas 1 a 3

MVP online
Home · Livro · Ferramentas (com cadastro) · Sobre · Blog (vazio). QR Codes funcionando. Formulário de lead ativo. E-mail de boas-vindas rodando.

### Fase 2 — Mês 2

Conteúdo e método
Página "O Método" publicada. 4 primeiros posts do blog (um por capítulo mais lido). Página de mentoria e treinamentos. WhatsApp integrado.

### Fase 3 — Mês 3+

Comunidade e escala
Área de comunidade para leitores. Sessões ao vivo mensais. Programa de afiliados para consultores. E-commerce de treinamentos online.
